# Output Format Standard

- User-facing response dùng tiếng Việt.
- Khi nội dung sử dụng tiếng Việt, phải viết tiếng Việt có dấu bằng UTF-8; không
  dùng tiếng Việt không dấu hoặc nội dung mojibake/sai mã hóa.
- Findings và verification evidence ưu tiên trước summary.
- File output trong protected path cần có xác nhận rõ ràng trước khi ghi.
