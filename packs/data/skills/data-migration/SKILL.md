# Data Migration

## Overview

Use this skill to produce evidence-based data migration plans with explicit
preconditions, rollout stages, validation queries, and rollback criteria.

## Process

1. Inspect the current and target data contracts.
2. Identify compatibility, volume, locking, and availability risks.
3. Design expand-and-contract stages where online migration is required.
4. Define backup, rollback, reconciliation, and observability checks.
5. Return executable verification steps.
