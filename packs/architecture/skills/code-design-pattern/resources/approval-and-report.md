# Approval And Report

Use this resource whenever the agent proposes or applies a design pattern.

For user-facing Vietnamese approval, completion, and no-pattern response templates, use [output-template-vi.md](output-template-vi.md).

## Approval Requirements

- Present the approval template before implementation.
- Include the proposed pattern group, specific pattern, rationale, and risks.
- Proceed only after explicit user approval.

## Final Report Requirements

- Use the completion template after implementation.
- Include files changed, verification evidence, and residual risks.
- Use the no-pattern template when a simpler solution is more appropriate than a design pattern.
