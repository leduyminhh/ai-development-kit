# Vietnamese Output Template

Use these templates for user-facing git workflow output.

## Before Commit

```text
Branch đề xuất:
Commit đề xuất:
Lý do chọn type/scope:
Files sẽ stage:
Commit body:
What changed:
- ...
  • ...

- ...

Why changed:
- ...
Important notes / breaking impact:
- ...
Verification:
```

## After Commit Or Push

```text
Branch:
Commit:
Push:
PR:
Verification:
Ghi chú:
```

## Changelog Or Release Notes

```text
Phạm vi:
Đối tượng đọc:
Tóm tắt:
Changelog / Release notes:
```
