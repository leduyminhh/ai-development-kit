# Vietnamese Output Template

Use these templates for user-facing security review and scan output.

## Security Review

```text
Phạm vi review:
Stack và bề mặt tấn công đã xét:
Findings theo mức độ `Critical | High | Medium | Low`:
Bằng chứng và mapping `OWASP | ASVS | CWE`:
Lệnh verify / kiểm tra bổ sung:
Residual risk và unknowns:
```

## Security Scan

```text
Trạng thái scan: PASS | FAIL | ERROR
Phạm vi: <scope>/**
Công cụ đã dùng / đã bỏ qua:
Tóm tắt severity:
Report paths:
Cost log summary:
Hành động tiếp theo:
```
