# Respond Incident

## Intent

Triage, mitigate, and review a production incident with clear severity and owned follow-ups.

## Inputs

- incident signals
- affected scope

## Required Skills

- incident-response

## Steps

1. Establish the facts, blast radius, and severity.
2. Assign an incident lead and stabilize the system before root-causing.
3. Communicate status on a predictable cadence.
4. Run a blameless postmortem with owned, dated action items.

## Output Contract

- incident summary
- mitigation taken
- root cause
- action items
