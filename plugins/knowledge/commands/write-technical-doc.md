# Write Technical Documentation

## Intent

Produce focused technical documentation aligned with the implemented system.

## Inputs

- documentation goal
- source scope

## Required Skills

- doc-write
- diagram-generate

## Steps

1. Inspect relevant source and existing documentation.
2. Select the smallest useful document structure.
3. Write flows, contracts, decisions, and diagrams.
4. Verify terminology and implementation alignment.

## Output Contract

- document summary
- technical content
- diagrams when useful
- verification notes
