# Deliver Feature

## Intent

Coordinate a bounded feature from planning through design, implementation, integration, review, testing, optional fixes, and release-readiness reporting.

## Inputs

- feature requirements
- source scope
- include extensions

## Required Skills

- java-implement
- python-implement
- react-implement

## Steps

1. Validate the feature goal, source scope, requested extensions, and starting evidence.
2. Run planning and required API or data design gates.
3. Route backend and frontend implementation through the matching canonical skills.
4. Integrate, review, test, and optionally fix the feature without bypassing blocked gates.
5. Summarize artifacts, verification evidence, blocked gates, and release readiness.

## Output Contract

- phase status
- artifacts
- verification
- blocked gates
- release readiness
