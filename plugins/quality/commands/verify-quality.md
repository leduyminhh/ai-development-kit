# Verify Quality

## Intent

Derive regression risks and verify the requested behavior.

## Inputs

- change scope
- acceptance criteria

## Required Skills

- test-qa-review
- test-automation-validate

## Steps

1. Identify behavior, boundaries, and regression risks.
2. Select unit, integration, and end-to-end checks.
3. Implement or run focused tests.
4. Report evidence and residual risk.

## Output Contract

- scenario summary
- executed tests
- failures
- residual risks
