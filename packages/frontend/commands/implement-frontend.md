# Implement Frontend

## Intent

Build a focused frontend change with accessibility and regression coverage.

## Inputs

- feature requirements
- design or API context

## Required Skills

- react-code-generate
- test-automation-validate

## Steps

1. Inspect the current component and state patterns.
2. Implement the smallest complete user flow.
3. Verify accessibility, loading, error, and empty states.
4. Run focused automated tests.

## Output Contract

- implementation summary
- changed behavior
- accessibility notes
- verification
